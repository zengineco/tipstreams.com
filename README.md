TipStreams is a static creator monetization studio for branded tip QR pages, multi-payment routers, and OBS overlays.

## Product stance

TipStreams is not a payment processor. It helps creators package wallets and payment links they already control into QR codes, permanent tip pages, and stream overlays.

The honest flagship is:

- 10 seconds for a single QR when the creator already has a wallet address or payment link.
- 10-20 minutes for a polished permanent tip page plus OBS overlay.

## Three tools

The homepage opens on one choice: pick a tool. Everything else is detail inside it.

1. QR Code — one branded QR for a single wallet or payment app. Fastest path.
2. Tip Page — one permanent link (tipstreams.com/p/you) with every method; update anytime, QR never changes. Includes the multi-payment router builder.
3. Overlay — drop your QR into an OBS-ready banner, sticker, corner badge, side panel, or alert card.

DogeFundMe is a slim cross-promo banner at the bottom — secondary ecosystem link, not part of the core flow.
